Files were downloaded from Google Fonts' CSS2 API (fonts.googleapis.com /
fonts.gstatic.com) as latin-subset WOFF2, then vendored into this repo at
src/design/fonts/ for offline-proof first paint (no runtime dependency on
Google Fonts).

Note on file count vs. weight count: Google Fonts serves each style
(normal/italic) of these families as a single variable-weight WOFF2; the
requesting browser resolves the CSS font-weight against the file's `wght`
fvar axis. That is why one physical .woff2 file legitimately backs multiple
discrete @font-face weight declarations in ../fonts.css — this mirrors the
exact @font-face structure Google's own CSS2 API returns for a modern
Chrome user agent, just pointed at local files instead of gstatic.com.
